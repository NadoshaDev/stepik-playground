## 1. Решение только через list

```python
target_animals = ['заяц', 'кот', 'слон']

filtered_animals = [
    animal for animal in animals
    if animal in target_animals
]

print(filtered_animals)
```

### Что я поняла:
- Это **list comprehension**
- Он сохраняет:
  - порядок
  - дубликаты
- Но проверка `in` работает медленнее, так как требует перебора элементов (линейный поиск)

---

## 2. Решение только через set

```python
target_animals = {'заяц', 'кот', 'слон'}

filtered_animals = {
    animal for animal in animals
    if animal in target_animals
}

print(filtered_animals)
```

### Что я поняла:
- Это **set comprehension**
- Он:
  - автоматически удаляет дубликаты
  - НЕ сохраняет порядок
- При формировании множества используется метод `.add()`, который добавляет элементы и автоматически исключает дубликаты
- Проверка `in` выполняется быстрее, так как не требует перебора всех элементов

---

## 3. Комбинированный подход (лучший баланс)

```python
target_animals = {'заяц', 'кот', 'слон'}

filtered_animals = [
    animal for animal in animals
    if animal in target_animals
]

print(filtered_animals)
```

### Ключевая идея:
- `set` → обеспечивает быструю проверку
- `list` → отвечает за хранение результата с сохранением порядка элементов

### Результат:
- высокая скорость
- сохранён порядок
- сохранены дубликаты

## Дополнительная задача: разделение данных на две группы

Я также решила связанную задачу — разделить данные на два списка.

### Вариант 1 — один проход (эффективный)

```python
filtered_animals = []
other_animals = []

selected_animals = {'заяц', 'кот', 'слон'}

for animal in animals:
    if animal in selected_animals:
        filtered_animals.append(animal)
    else:
        other_animals.append(animal)

print(filtered_animals)
print(other_animals)
```

### Что я поняла:
- Только **один проход** по списку
- Более эффективно для больших данных
- Полный контроль над логикой

---

### Вариант 2 — list comprehension (чисто, но два прохода)

```python
selected_animals = {'заяц', 'кот', 'слон'}

filtered_animals = [animal for animal in animals if animal in selected_animals]
other_animals = [animal for animal in animals if animal not in selected_animals]

print(filtered_animals)
print(other_animals)
```

### Что я поняла:
- Код короче и чище
- Но список обрабатывается **два раза**
- Менее эффективно на больших объёмах данных

---

## Ключевой вывод из сравнения

Не существует одного "лучшего" решения — всё зависит от задачи.

```text
Эффективность → используем один цикл (for)
Читаемость → используем list comprehension
```

## Важное открытие: разница между set и dict

В процессе работы я столкнулась с неожиданным моментом: фигурные скобки в Python используются не только для `set`, но и для `dict`.

```python
{} → это dict (а не set)
```

### Примеры:
```python
a = {}              # словарь (dict)
a = set()           # пустое множество (set)
a = {"кот", "заяц"}  # множество (set)
a = {"кот": 1}      # словарь (dict)
```

### Ключевое правило:
- есть `:` → это dict
- нет `:` → это set
- пустые `{}` → всегда dict

# 🥒 Овощное Комбо

Небольшая консольная игра на Python.

Игрок случайным образом собирает набор овощей. После каждого выпадения программа показывает, какой овощ был получен, а в конце выводит весь собранный набор.

Но собрать овощи недостаточно. Настоящие мастера знают секретные комбинации!

## Комбо №1
- 2 огурца
- 1 помидор

## Комбо №2
- 1 огурец
- 1 помидор
- 1 перец
- 1 лук

Если в собранном наборе присутствует хотя бы одно из комбо, игрок получает звание Повелителя Овощей.

## Что я отработала
- Работа со списками (`list`)
- Добавление элементов через `append()`
- Подсчёт элементов через `count()`
- Сложные условия с `and` и `or`
- Булевы переменные (`first_combo`, `second_combo`)
- Разделение большой проверки на понятные логические части

## Мой ход мыслей
Сначала я пыталась искать комбо как готовый список внутри списка овощей. Позже поняла, что комбо — это не отдельный объект, а набор требований к содержимому. После этого решение стало намного проще и понятнее.

Это была одна из первых задач, где я начала активно использовать булевы переменные для описания игровой логики.

##  Код
```python
import random

ingredient = ['огурец', 'помидор', 'лук', 'перец']
my_vegetables = []

for _ in range(6):
    vegetable = random.choice(ingredient)
    my_vegetables.append(vegetable)
    print("Вы получили:", vegetable)

print("Мы собрали:", my_vegetables)

cucumber_count = my_vegetables.count('огурец')
tomato_count = my_vegetables.count('помидор')
pepper_count = my_vegetables.count('перец')
onion_count = my_vegetables.count('лук')

first_combo = cucumber_count > 1 and tomato_count > 0
second_combo = (
  cucumber_count > 0 and tomato_count > 0
  and pepper_count > 0 and onion_count > 0
)

if first_combo or second_combo:
    print("О великий игрок, вы повелитель овощей!")
else:
    print("Увы, вам не повезло!")

# 🔤 Acronym Challenge

Небольшая консольная игра на Python.

Программа случайным образом генерирует аббревиатуру из трёх букв русского алфавита. Задача игрока — придумать три слова так, чтобы первые буквы этих слов совпали с выпавшей аббревиатурой.

Например, если программа сгенерировала:

```text
КБС
```

Игрок может ввести:

```text
Кот
Белка
Самолёт
```

Если первые буквы совпадают с аббревиатурой, игрок побеждает.

## Что я практиковала

- Работа со строками
- Получение первого символа строки
- Изменение регистра через `upper()`
- Генерация случайных символов через `random.choice()`
- Работа с циклами
- Постепенная сборка строки из пользовательского ввода

## Интересная часть проекта

Первую версию задачи я решила через три отдельные проверки для каждого слова.

Позже я переписала решение так, чтобы программа собирала пользовательскую аббревиатуру в цикле и сравнивала уже готовый результат. Такой подход делает код короче, чище и проще для расширения.

## Код

```python
import random

abbreviation = ''

for _ in range(3):
    generated_letter = random.choice("АБВГДЕЖЗИКЛМНОПРСТУФХЦЧШЩЫЭЮЯ")
    abbreviation += generated_letter

print(abbreviation)

player_abbreviation = ''

for _ in range(3):
    player_word = input().strip()
    player_abbreviation += player_word[0].upper()

if player_abbreviation == abbreviation:
    print("О великий игрок! Это знал 1% россиян!")
else:
    print("ДВО - даже великие ошибаются")
```

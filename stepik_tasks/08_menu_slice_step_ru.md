
# Задача 08 — Меню для чётных и нечётных месяцев

## Что делает программа
Эта программа формирует разное меню ресторана в зависимости от номера месяца.

Для чётных месяцев выбираются блюда с чётными индексами.
Для нечётных месяцев — блюда с нечётными индексами.

## Что я отработала
Основная тема задачи — срезы списков (list slicing).

Раньше я видела простые срезы, но впервые работала с конструкциями вида:
`menu[0::2]` и `menu[1::2]`

Я поняла, что третий параметр в срезе — это шаг, который позволяет выбирать каждый второй элемент из списка.

## Момент «Ага!» (Инсайт)
Сначала я решила задачу через обычный `if`:

```python
if month % 2 != 0:
    result = menu[1::2]
else:
    result = menu[0::2]
```

Позже увидела решение другого студента:
```python
menu[month % 2::2]
```

Моя первая реакция была:
> «Что это вообще значит?» 😄

Разобрав это по шагам, всё вдруг встало на свои места.

Я осознала, что:
- `month % 2` возвращает `0` для чётных месяцев;
- `month % 2` возвращает `1` для нечётных месяцев.

Это значение можно сразу использовать как стартовый индекс среза.

Для меня это был один из тех маленьких трюков Python, которые сначала выглядят загадочно, но становятся элегантными, как только понимаешь логику.

## 🐍 Код

### мой первый вариант (через if/else)
```python
menu = ['Борщ', 'Салат Цезарь', 'Пицца Маргарита', 'Картофель фри',
        'Стейк', 'Рататуй', 'Чизкейк', 'Тирамису']

month = int(input())                   # Номер месяца

if month % 2 != 0:
    result = menu[1::2]
else:
    result = menu[0::2]

print("Сегодня в меню:", result)
```

### Мой второй вариант (элегантный срез)
```python
menu = ['Борщ', 'Салат Цезарь', 'Пицца Маргарита', 'Картофель фри',
        'Стейк', 'Рататуй', 'Чизкейк', 'Тирамису']

month = int(input())                   # Номер месяца

result = menu[month % 2::2]

print("Сегодня в меню:", result)
```

## Задача
Вывести буквы из списка от start до stop с шагом 1 или -1.

## Моё решение

**1. Отрицательные индексы → положительные**

```python
if start < 0:
    start = len(letters) + start
```
    
Хочу понимать механику, а не полагаться на магию Python.
**2.Расчёт границы end**

```python
if step == 1:
    end = stop + 1
elif step == -1:
    end = stop - 1
```
   
range() не включает правую границу. Для прямого прохода прибавляю 1, для обратного — отнимаю 1.
**3.Цикл**

```python
for i in range(start, end, step):
    print(letters[i])
```

## Что добавила от себя
Защита от плохого шага:

```python
else:
    print("ошибка")
    exit()
```
    
Если введут шаг 0 или 2 → программа не сломается, а скажет "ошибка". Код должен быть устойчивым!

## Вывод
Разделила задачу на части: нормализация → расчёт границ → цикл. Добавила проверку шага — это мой стандарт качества.

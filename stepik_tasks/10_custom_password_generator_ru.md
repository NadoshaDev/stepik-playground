# 🔐 Custom Password Generator

Консольный генератор паролей на Python.

Пользователь может самостоятельно настроить состав будущего пароля. По умолчанию используются только строчные английские буквы, но при желании можно добавить цифры и специальные символы.

После выбора параметров программа создаёт случайный пароль длиной 8 символов.

## Настройки

Пользователь отвечает на два вопроса:

* Добавлять ли цифры?
* Добавлять ли специальные символы?

В зависимости от ответов программа формирует набор допустимых символов и генерирует пароль.

## Что я практиковала

* Работа со строками
* Постепенная сборка строки
* Использование `random.choice()`
* Обработка пользовательского выбора
* Упрощение условий
* Построение набора символов через `+=`

## Особенность решения

Первоначально решение содержало несколько больших условий для всех возможных комбинаций ответов пользователя. Позже я упростила логику и начала постепенно расширять набор допустимых символов через `+=`.

Так код стал короче, чище и легче для чтения.

---

## 🐍 Код
```python
import random

letters = 'abcdefghijklmnopqrstuvwxyz'
numbers = '0123456789'
special_symbols = '*$@(!'                  
password = ''
allowed_characters = ''

use_numbers = input().strip().lower()            # Хочешь, чтобы в пароле могли быть цифры - да/нет
use_special_symbols = input().strip().lower()    # Хочешь, чтобы в пароле могли быть спец символы - да/нет

if use_numbers == 'да': 
    allowed_characters += numbers
    
if use_special_symbols == 'да':
   allowed_characters += special_symbols
    
for _ in range(8):
    password += random.choice(letters + allowed_characters)

print("Ваш случайный пароль:", password)

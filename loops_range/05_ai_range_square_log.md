4. Собираем в новый список


##  Эволюция подхода: Цикл vs List Comprehension

Для закрепления я прогнала одну и ту же задачу двумя способами:

**1. Классический цикл (тренировка):**
```python
filtered_numbers = []
for number in numbers:
    if number % 2 == 0:
        filtered_numbers.append(number ** 2)
```

**2. List Comprehension (оптимизация):**
```python
filtered_numbers = [number ** 2 for number in numbers if number % 2 == 0]
```

** Вывод:**  
Оба варианта дают одинаковый результат, но второй короче, читается как одно предложение и выполняется быстрее. Это подтвердило, что `list comprehension` — не просто «синтаксический сахар», а инструмент для ясного и поддерживаемого кода.
